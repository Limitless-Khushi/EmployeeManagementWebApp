## Project 1: Employee Management Web App
A comprehensive full-stack ASP.NET Core MVC application developed to streamline organizational workforce management. This system provides complete CRUD functionality for handling employee digital records efficiently.

### Project Scope & Features
* **Full CRUD Lifecycle:** Create, Read, Update, and Delete employee profiles dynamically.
* **Data Persistence:** Backed by MS SQL Server via Entity Framework Core using Code-First Migrations.
* **Server-Side Validation:** Form verification built-in with strongly typed Data Annotations.
* **Advanced Interface:** Styled cleanly utilizing responsive Bootstrap dark elements with active metrics track and live client-side search.

### Tech Stack
* **Backend:** ASP.NET Core MVC (.NET 8.0), C#
* **Database O/RM:** Entity Framework Core
* **Database Engine:** Microsoft SQL Server (LocalDB)
* **Frontend:** Razor Views, HTML5, CSS3, Bootstrap 5, JavaScript
